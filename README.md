# Mobile Sales Data Analysis & Dashboard

This project analyzes a mobile phone sales dataset to uncover sales patterns and key insights using Microsoft Excel.

##  Dataset
The dataset contains mobile phone sales records with fields such as:
- **Price_USD** – price of the mobile unit
- **Units_Sold** – number of units sold
- **Revenue_USD** – total sales revenue
- **Customer_Rating**
- **Sale_Month**, **Month**, **Sale_Year**

(Source: Kaggle mobile sales dataset)

##  What I Did
1. Cleaned and prepared the data in Excel.
2. Converted numeric months into month names for better readability.
3. Built PivotTables and charts to analyze:
   - Total revenue by brand
   - Monthly sales trends
   - Price distribution
   - Top 5 best‑selling brands
4. Created an interactive dashboard with charts and filters.

##  Key Insights
- Most revenue comes from a few top brands.
- Certain months show peak sales.
- Phones in mid‑price range have highest sales frequency.

##  Excel Dashboard
Included:
- PivotTables
- Revenue trend line chart
- Bar chart of price distribution
- Filter panels for brands and months
